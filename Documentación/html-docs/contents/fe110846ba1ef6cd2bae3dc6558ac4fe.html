Creado por: Solis Patricia Fecha de Creación: 07/11/2022

Actualizado por: Solis Patricia Fecha de Actualización: 14/11/2022

Actores: Administrador

Disparador: Selección de opción "Modificar Combo"

Descripción: El administrador ingresa a ésta sección para Modificar Combo de la base de datos.

Pre-condición: El administrador debe ingresar a gestionar combo y seleccionar la opción -Modificar combo

Post-condición: Al MODIFICAR un combo, se deberá especificar Tipo de productos que pertenecen al combo y cantidad para determinar su stock

Flujo Normal

El administrador seleciona "MODIFICAR Combo". El sistema consulta con la base de datos. El sistema muestra en pantalla los distintos tipos de combos y productos incluidos en el que puede MODIFICAR : Vuelos, excursiones, hospedaje, Otros... El administrador selecciona el producto deseado para generar la modificacion del combo El sistema consulta con la base de datos. El sistema muestra en pantalla los resultados de los distintos productos encontrados con opción de agregar al combo o eliminar del combo, y MODIFICA a cada campo de los resultados. El administrador realiza los cambios deseados y selecciona ACEPTAR. El sistema muestra en pantalla un mensaje : "usted ha MODIFICADO XCombo-, está seguro?" con la opciones de "CANCELAR" o "APLICAR". El administrador CANCELA. El sistema vuelve a mostrar los resultados obtenidos sin modificar, esperando la acción del administrador. El administrador realiza los cambios deseados y selecciona ACEPTAR y APLICA El sistema guarda los cambios realizados en la base de datos. El sistema pregunta: "SEGUIR MODIFICANDO" O "VOLVER A MENÚ GESTIONAR COMBO".

Flujo Alternativo

2. El administrador selecciona la opción "VOLVER AL MENÚ GESTIONAR COMBO".

El sistema redirige al menú -Gestionar combo a la espera de la selección de alguna opción por parte del administrador.
