16- Consultar MU Cliente

UMLUseCase
Travel Easy :: Modelado Travel Easy :: Travel Easy :: 16- Consultar MU Cliente

Description

Creado por: Patricia Illbele Fecha de Creación: 12/11/2022 Actualizado por: Fecha de Actualización: Actores: Google, Administrador, Cliente y Usuarios externos. Disparador: Necesidad de consultar manual de usuario administrador. Descripción: Caso de Uso en donde se consulta el manual de usuario para el perfil administrador Pre-condición: El usuario debe contar con perfil cliente. Post-condición: Ninguna.

Flujo Normal

El usuario se loguea ingresando nombre y contraseña El sistema verifica tipo de usuario (Cliente, Administrador). El usuario accede al manual.

Flujo Alternativo

1.El usuario no tiene cuenta. El sistema devuelve un formulario en donde requiere datos a completar por el usuario, para crear la cuenta. El usuario completa datos requeridos por el sistema y confirma la carga. El sistema envía un mail de confirmación a la casilla del usuario para verificar la cuenta. El usuario accede al mail y confirma la creación de cuenta mediante ingreso a link proporcionado. Prosigue en punto 1 del flujo normal 2. El usuario no ingresa datos personales obligatorios correctos. El sistema envía un mensaje recordando ingresar los datos obligatorios. Prosigue punto 4 del flujo normal.

  1. El usuario no es administrador. El sistema no permite acceso al manual de administrador.

  2. El usuario no recuerda contraseña El usuario completa formulario para recordar contraseña. El sistema envía mail a la casilla registrada para reestablecer contraseña. Prosigue en punto 1 del flujo normal.
