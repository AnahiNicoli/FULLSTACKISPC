  • Actores: Administrador

  • Disparador: Selección de opción "Modificar Productos"

  • Descripción: El administrador ingresa a ésta sección para Modificar productos de la base de datos.

  • Pre-condición: El administrador debe ingresar a gestionar productos y seleccionar la opción -Modificar productos

  • Post-condición: Al MODIFICAR un producto, se deberá especificar Tipo de producto y cantidad para determinar su stock

Flujo Normal

  1. El administrador selecciona la opción de modificar un producto existente.
  2. El sistema busca los datos del producto seleccionado.
  3. El sistema muestra un formulario para modificar el producto.
  4. El administrador realiza las modificaciones pertinentes en el producto.
  5. El administrador resgualda los cambios realizados en el producto.
  6. El sistema valida que los campos obligatorios estén completos.
  7. El sistema actualiza el producto en la base de datos.
  8. El sistema muestra un mensaje de confirmación de que el producto se ha actualizado correctamente.

Flujo Alternativo

. Si el administrador desea cancelar la modificación del producto:

  1. El administrador hace clic en el botón "Cancelar".
  2. El sistema descarta la modificación del producto y regresa a la lista de productos existentes.
