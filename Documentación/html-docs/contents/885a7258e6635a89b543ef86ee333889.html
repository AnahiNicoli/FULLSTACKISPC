Description

Nombre de cao de uso: Eliminar combo Creado por: Solis Patricia Fecha de Creación: 07/11/2022

Actualizado por: Solis Patricia Fecha de Actualización: 14/11/2022

Actores: Administrador

Disparador: Selección de opción "Eliminar Combo"

Descripción: El administrador ingresa a ésta sección para Eliminar combo a la base de datos.

Pre-condición: El administrador debe ingresar a gestionar combo y seleccionar la opción -Eliminar Combo

Post-condición: Al eliminar un combo, se deberá especificar Tipo de combo

Flujo Normal

El administrador seleciona "Eliminar Combo". El sistema muestra en pantalla los distintos tipos de Combo que puede eliminar. El administrador selecciona el combo deseado. El sistema muestra en pantalla los distintos combos y pregunta cual desea eliminar. El administrador una vez completado, acepta y aplica los cambios. El sistema muestra en pantalla un mensaje con un resumen de la acción Ej: "usted ha eliminado XCombo-, está seguro?" con la opciones de "Modificar" o "Aplicar". El administrador Aplica los cambios a los combos para eliminarlos de la base de datos.

Flujo Alternativo

2.

El sistema muestra un formulario para completar: Tipo de combo El administrador completa el formulario con todos los campos requeridos y acepta. El sistema muestra en pantalla un resumen de lo completado, y pide la opción de "MODIFICAR" o "APLICAR". El administrador en caso de selecciona "MODIFICAR". El sistema regresa al estado anterior, para continuar nuevamente con el flujo alternativo. El administrador selecciona "APLICAR". El sistema muestra en pantalla el mensaje "Eliminado con éxito" y pregunta si "Eliminar otro combo" o "Volver a la pantalla
