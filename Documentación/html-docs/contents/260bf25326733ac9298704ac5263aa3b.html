  • Actores: Administrador

  • Disparador: Selección de opción "Crear Productos"

  • Descripción: El administrador ingresa a ésta sección para CREAR productos a la base de datos.

  • Pre-condición: El administrador debe ingresar a gestionar productos y seleccionar la opción -Crear productos

  • Post-condición: Se crea un nuevo producto en la base de datos de forma exitosa

Flujo Normal

  1. El administrador selecciona la opción de crear una nuevo producto.
  2. El sistema muestra un formulario para crear un nuevo producto, que incluye campos para el nombre del producto.
  3. El administrador ingresa los detalles del nuevo producto.
  4. El administrador guarda el producto nuevo.
  5. El sistema valida que los campos obligatorios estén completos.
  6. El sistema almacena al producto nuevo en la base de datos.
  7. El sistema muestra un mensaje de confirmación de que el producto nuevo se ha creado correctamente.

Flujo Alternativo

. Si el administrador desea cancelar la creación del producto nuevo:

  1. El administrador hace clic en el botón "Cancelar".
  2. El sistema descarta el trabajo y regresa a la lista de producto existentes.
